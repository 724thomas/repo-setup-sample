이 프로젝트는 Github에서 Repository를 사용할 때 관리해야 할 것들을 연습하기 위한 repository 입니다.

[See Demo](https://github.com/724thomas/repo-setup-sample)

## Prerequisite

- node.js: ^11.0
- java: ^8

## How to start

'''shell
$ git clone {repo addr}
$ cd repo-setup-sample
$ npm install
$ npm start
'''

## Features

- Blog
- Reply
- My page
- Donate with bitcoin
